title: "KNX (Gateway)"
description: "KNX bina otomasyonu — Gateway uygulaması ile entegrasyon"

  order: 14

KNX protokolü, inSCADA platformuna doğrudan dahil değildir. Ayrı bir **KNX Gateway** uygulaması aracılığıyla entegre edilir. Gateway, KNX cihazlarından veri toplayarak inSCADA'ya REST API üzerinden aktarır.
