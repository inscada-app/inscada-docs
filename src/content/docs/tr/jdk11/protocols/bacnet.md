sidebar:
  order: 13
---

BACnet protokolü, inSCADA platformuna doğrudan dahil değildir. Ayrı bir **BACnet Gateway** uygulaması aracılığıyla entegre edilir. Gateway, BACnet IP cihazlarından veri toplayarak inSCADA'ya REST API üzerinden aktarır.
